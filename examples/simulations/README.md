# Scenario Resources

This folder contains reusable simulation JSON configurations grouped by domain.

- `legacy/`: early baseline scenarios and chart-related examples
- `cryptokitties/`: CryptoKitties-based scenarios
- `ether-did/`: Ether DID lifecycle scenarios
- `dao/`: DAO-focused scenarios used for the DAO study and plots

Use these files as templates or as import sources from the client.

Keep only simulation JSON files here. Backend runtime resources, papers, images, and generated outputs should stay elsewhere.
